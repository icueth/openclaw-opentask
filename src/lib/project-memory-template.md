# Project Memory

## Overview
This file is the project's long-term memory. Every task MUST read this file before starting and update it after completing.

## Memory Format

Each entry should follow this format:

```
## YYYY-MM-DD - Task Title
- **Task ID**: task-id-here
- **Status**: completed/failed
- **Summary**: Brief description of what was done
- **Files Created**: list/of/files.js
- **Files Modified**: list/of/files.js
- **Key Learnings**: Important insights or decisions
- **Next Steps**: What should be done next (if applicable)

---
```
